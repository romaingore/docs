**Ops** (Operations) désigne l'ensemble des pratiques liées au déploiement, à la gestion et à la surveillance des applications et infrastructures.

## DevOps en bref

Le **DevOps** est une culture qui réunit le développement (Dev) et les opérations (Ops) pour livrer plus vite et plus fiablement.

```
Plan → Code → Build → Test → Release → Deploy → Operate → Monitor
 └──────────────────────────────────────────────────────────────┘
```

## Concepts clés

| Concept | Description |
|---|---|
| **CI/CD** | Intégration et déploiement continus |
| **IaC** | Infrastructure as Code (Terraform, Ansible) |
| **Conteneurisation** | Isoler les applications (Docker) |
| **Orchestration** | Gérer les conteneurs à l'échelle (Kubernetes) |
| **Monitoring** | Surveiller les performances et la santé |
| **Cloud** | Héberger sur des infrastructures distantes |
